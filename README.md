# ☕ Maratona Java - DevDojo

Este repositório contém meus estudos e exercícios realizados durante a **Maratona Java do DevDojo**, ministrada pelo canal DevDojo no YouTube.

O objetivo é documentar minha evolução na linguagem Java, desde os fundamentos até conceitos mais avançados, praticando lógica de programação, orientação a objetos e boas práticas de desenvolvimento.

---

## 🚀 Objetivo

- Aprender Java do zero ao avançado
- Desenvolver lógica de programação
- Praticar resolução de exercícios
- Construir base sólida para desenvolvimento backend
- Evoluir para desenvolvimento com Spring Boot futuramente
